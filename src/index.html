Teste para a vaga de desenvolvedor de Interface

Olá, seja bem vindo ao teste para a vaga desenvolvedor de Interface da Ignição Digital!

O teste é composto por 5 questões que encontram-se nos respectivos diretórios.

Sinta-se livre para utilizar bibliotecas que estiver acostumado, criar estrutura de diretórios, editar e renomear arquivos e utilizar linguagem de backend se necessário.

Bibliotecas JS são permitidas em todos os testes exceto o t3 e t5.

No teste 2 evite o uso de templetes html/css prontos, gostariamos de ver como você estrutura o html

Tudo que você precisa saber sobre cada questão se encontra no console do navegador quando aberto o index.html de cada teste

O que será avaliado nesse teste?

  • Organização/qualidade de código/Semântica, legibilidade
  • Capacidade técnica
  • Conhecimentos de tecnologias front end em geral
    • Design patterns
    • Programação OO
    • Arquitetura CSS
  • Atenção
  • Dedicação
  • Agilidade
  • Desempenho do código
